HTML

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menata konten pada halaman web. HTML berfungsi sebagai kerangka dasar sebuah situs web, mendefinisikan struktur dan isi seperti teks, gambar, dan tautan menggunakan serangkaian kode atau tag. Browser web kemudian membaca kode HTML ini untuk menampilkan halaman web kepada pengguna.
HTML
•Membuat Struktur Halaman Web: Mendefinisikan bagian-bagian utama seperti header, body, dan footer pada sebuah halaman.
•Menampilkan Konten: Mengatur teks, gambar, video, tabel, dan elemen multimedia lainnya agar tampil sesuai dengan yang diinginkan.
•Menghubungkan Halaman (Hyperlinks): Menggunakan kode khusus untuk membuat tautan yang memungkinkan pengguna berpindah dari satu halaman ke halaman lain di internet.
•Mendukung SEO: Membantu pemilik website mengarahkan pengguna dan pembaca ke halaman yang relevan melalui hyperlink.
